# Image Requirements

This document lists all images needed for the portfolio website.

## Hero Section
- `hero/heroImg1.jpg` - Main hero image (animated picture/character)
  - Recommended size: 800x800px or larger (square format)
  - Format: JPG, PNG, or WebP

Currently, the About section uses CSS-based 3D graphics. If you want to add images later:
- `about/aboutImg1.jpg` - About section image 1
- `about/aboutImg2.jpg` - About section image 2
  - Recommended size: 800x600px or larger
  - Format: JPG, PNG, or WebP

## Notes
- All images should be optimized for web (compressed but maintaining quality)
- Use descriptive alt text in the components
- Images are imported as ES6 modules, so they will be processed by Vite's build system
- If an image fails to load, fallback styles are implemented in the components
