# Images Assets

This folder contains all image assets organized by section:

- `hero/` - Hero section images
- `customers/` - Customer section scrolling images
- `projects/` - Project showcase images
- `about/` - About section images (if needed)

## Naming Convention

- Hero images: `heroImg1.jpg`, `heroImg2.jpg`, etc.
- Customer images: `customerImg1.jpg`, `customerImg2.jpg`, etc.
- Project images: `projectImg1.jpg`, `projectImg2.jpg`, etc.
- About images: `aboutImg1.jpg`, `aboutImg2.jpg`, etc.
